About This Site

This site exists to promote ideas and direct action for an autonomous society. For an introduction to A or B visit our About page. For information about our program visit our unifying document “As We See It“.

We built this website using decentralized consensus and open source digital technology. This includes the InterPlanetary File System and Ethereum blockchain. This technology is used to promote peer-to-peer relations (disintermediation), censorship resistance (immutability), self-organization and direct democracy.
